This is a demo of the new HTML5 localStorage API in action
You have entered: 
window.localStorage['data'] = "hello world";
return window.localStorage['data'];

This demo works in the following browsers:

  • Firefox 3.5+
  • Internet Explorer 8+
  • Chrome 5+
  • Safari 4+
  • Opera 10.5+

This code is available on github
